# Defend Your Castle Script

A robust automation script for the Roblox game "Defend Your Castle" using the Rayfield UI library.

## Features
- **Auto Buy Defense**: Automatically upgrades your castle defenses.
- **Auto Roll Enchant**: Continuous enchanting of items.
- **Auto Start Challenge**: Keeps challenges active.
- **Auto Request Enemies**: Farms enemies efficiently.

## How to use
Copy the contents of `script.lua` into your preferred Roblox executor.
